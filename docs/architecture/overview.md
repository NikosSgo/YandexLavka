Проект состоит из следующих микросервисов:

- ApiGateway - API Gateway на базе YARP, маршрутизирует запросы к микросервисам
- UserService - Сервис управления пользователями
- WareHouseService - Сервис управления складом, заказами и товарами
- DeliveryService - Сервис управления доставкой
- AuthService - Сервис авторизации
- Frontend - Клиентское React приложение

Действия UserService:

- Работа с профилем (Создать, прочитать, изменить, удалить)
- Добавить товар в избранное
- Добавить адрес
- Инициализовать заказ
- Посмотреть все заказы
- Отменить заказ

База данных UserService должен содержать:

- Профиль
- Адреса
- Покупки
- Избранное

Действия WareHouseService (Надо бы поделить на сервисы):

- Посмотреть ассортимент товаров
- Инициализовать корзину
- Добавить товар в корзину
- Удалить товар из корзины
- Добавить товар в ассортимент
- Убрать товар из ассортимента
- Инициализовать заказ из корзины
- Добавить товар в заказ
- Убрать товар из заказа
- Передать заказ в доставку
- Отменить заказ

WareHouseService должен содержать:

- Ассортимент склада
- Корзины склада
- Заказы склада

Действия DeliveryService:

- Создать профиль доставщика
- Принять заказ
- Принять заказ в руки
- Отменить доставку
- Подтвердить доставку

DeliveryService должен содержать:

- Профиль
- Принятые заказы

Действия AuthService:

- Регистрация
- Вход
- Полученние токена

AuthService должен содержать:

- Данные аккаунта
